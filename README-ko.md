## 🚀 전문적인 커뮤니티 관리를 위한 플랫폼
**Densory**는 성장하는 서버 프로젝트를 위한 최고의 올인원 솔루션입니다. 우리는 여러분이 커뮤니티를 완벽하게 통제할 수 있도록 설계된 고성능 **Discord 봇 시스템** 및 **FiveM 스크립트**를 개발합니다.

중재, 보안, 복잡한 로깅, 게임 내 기능 등 모든 기술은 우리가 제공하며, 관리의 주도권은 여러분에게 있습니다. 모든 기능은 중앙 집중식 **Densory 컨트롤 센터(Densory Control Center)**를 통해 관리됩니다.

### ✨ 왜 Densory인가요? (차별화된 특징)

| 특징 | 설명 |
| :--- | :--- |
| 🏷️ **100% 화이트 라벨** | 브랜드명, 로고, 색상까지 모두 여러분의 것으로 설정 가능합니다. 외부에서는 여러분이 직접 개발한 것처럼 보입니다. 임베드 메시지에 불필요한 타사 광고가 포함되지 않습니다. |
| 🖥️ **유연한 호스팅 선택** | **100% 셀프 호스팅:** 자체 서버에서 봇과 스크립트를 직접 호스팅하여 데이터를 완벽하게 관리하세요. <br>**Densory 관리형 호스팅:** 서버 관리가 번거로우신가요? 99% 가동률을 보장하는 당사의 옵션 호스팅 서비스를 이용하세요. |
| ⚡ **즉시 활성화** | 대기 시간 없음. 결제(PayPal 또는 암호화폐)가 완료되면 모든 디지털 상품과 라이선스가 대시보드에서 즉시 활성화됩니다. |
| 🎁 **무료 체험** | 직접 경험해 보세요: 결제 정보를 입력할 필요 없이 일부 제품에 대한 무료 체험 라이선스를 제공합니다. |

## 🛠️ 프리미엄 핵심 제품

### 🤖 Discord 봇 시스템
생성된 즉시 사용 가능한 당사의 봇은 전문적인 Discord 서버의 모든 영역을 지원하며 단 몇 분 만에 도입할 수 있습니다.

*   🛡️ **보안 및 안티-누크(Anti-Nuke):** 서버를 레이드 공격으로부터 보호하세요. 피싱 도메인 탐지(Threat-Feed), 자동 중재 및 위기 대응 시스템을 포함합니다.
*   🔐 **인증 및 본인 확인:** 커뮤니티를 위한 가장 안전한 게이트웨이. 부계정 탐지(브라우저 핑거프린팅), VPN 차단 및 캡차(Captcha) 통합 기능을 제공합니다.
*   📊 **로깅 및 분석:** 모든 상황을 한눈에 파악하세요. 상세한 감사 로그, 서버 통계, 실시간 지표 및 외부 시스템(FiveM 등)을 Discord에 직접 기록하는 사용자 지정 API를 지원합니다.
*   💬 **Discord 관리자:** Densory 대시보드에서 서버를 직접 관리하세요! 임베드 메시지 전송, 실시간 채팅 읽기, 메시지 분석 및 자동화된 스케줄 생성(Automations)이 가능합니다.

### 🏎️ FiveM 스크립트
최고의 성능을 위해 개발되었습니다. 당사의 스크립트는 기존 프레임워크(ESX / QBCore)와 원활하게 통합되며, 서버에 깊이 있고 리소스 친화적인 기능을 확장합니다. Densory 대시보드와의 직접 연결을 통해 게임 내 지표와 로그를 실시간으로 모니터링할 수 있습니다.

## ⚙️ Densory 컨트롤 센터
서버 인프라의 심장부입니다. 현대적이고 안전한 웹 인터페이스를 통해 다음 기능을 제공합니다:
- **라이선스 관리:** 제품 기간 연장, 업그레이드, 혹은 타 서버로 라이선스 이전.
- **팀 협업:** 팀 채팅 통합 및 설정 공유(공유 코드를 통한 제품 설정 공유).
- **안전한 결제:** PayPal 및 암호화폐(Bitcoin, Ethereum, Solana 등)를 통한 투명한 정산.
